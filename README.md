# ReactMediaPlayer

ReactMediaPlayer pings the Youtube API, allows you search youtube videos, and displays five videos relevant to the search term. Videos are also embedded.

![search_video](/assets/search-example-gif.gif)

### Technologies
- React
- Youtube API
- Babel
- Webpack
- Lodash
- CSS3
- HTML5
